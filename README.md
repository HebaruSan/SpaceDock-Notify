SpaceDock's HTTP notifier for CKAN. Seperate project as it depends on NetKAN-meta.  
  
Setup (after cloning):  
  
git clone https://github.com/KSP-CKAN/NetKAN  
virtualenv -p /usr/bin/python3 .  
. bin/activate  
pip install -r requirements.txt  
python app.py  
(edit config.ini)  
python app.py
